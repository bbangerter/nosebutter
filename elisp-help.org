** Using a ~lambda~ function with ~mapcar~
   Executing the code below:
   
    #+begin_src elisp :exports both
    (mapcar (lambda (x) (format "%s world!" x)) '("Hi" "Hello" "Howdy"))
    #+end_src


   Returns the following results:
    #+results:
    : ("Hi world!" "Hello world!" "Howdy world!")
